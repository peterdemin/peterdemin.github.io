My Radicle JourneyÂ¶

Radicle is an open source, peer-to-peer code collaboration stack built on Git. Unlike centralized code hosting platforms, there is no single entity controlling the network. Repositories are replicated across peers in a decentralized manner, and users are in full control of their data and workflow.

We will initialize Radicle on MacOS building it from source.

Of course, you can do curl | bash, but I manage most of software on my macbook through brew, so Iâ€™ll use that. Note, however, that you should never install Rust through brew directly, as this is a recipe for disaster.
